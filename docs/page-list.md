# 运维平台前端页面清单

## 1. 页面结构

一期前端建议按下面菜单组织：

1. 登录
2. Portal
3. 云账号管理
4. 域名管理
5. 主机管理
6. CMDB
7. JumpServer
8. 账单分析
9. 任务中心
10. 审计日志

## 2. 页面清单

### 2.1 登录页

路径建议：

- `/login`

主要功能：

- 用户登录
- 登录态校验
- 登录失败提示

### 2.2 Portal 首页

路径建议：

- `/portal`

主要模块：

- 资产统计卡片
- DNS 统计卡片
- 云主机统计卡片
- 本月费用统计卡片
- 最近任务列表
- 同步失败和异常提示

### 2.3 云账号列表页

路径建议：

- `/accounts`

主要功能：

- 账号列表
- 按 provider 筛选
- 新增账号
- 编辑账号
- 启用/停用账号
- 测试连接
- 手动同步

### 2.4 域名列表页

路径建议：

- `/domains/zones`

主要功能：

- 域名列表
- provider 筛选
- 账号筛选
- 查看域名详情
- 手动同步域名数据

### 2.5 DNS 记录列表页

路径建议：

- `/domains/records`

主要功能：

- 记录列表
- 域名筛选
- 类型筛选
- 状态筛选
- 批量启用/停用
- 批量修改 TTL
- 导出记录

### 2.6 DNS 记录详情页

路径建议：

- `/domains/records/:id`

主要功能：

- 查看记录详情
- 查看来源平台字段
- 查看变更审计

### 2.7 主机列表页

路径建议：

- `/assets/instances`

主要功能：

- 主机列表
- provider / 账号 / 区域 / 标签筛选
- 查看 JumpServer 状态
- 手动同步主机数据
- 跳转主机详情

### 2.8 主机详情页

路径建议：

- `/assets/instances/:id`

主要功能：

- 基础信息展示
- 网络信息展示
- 标签展示
- 同步记录展示
- JumpServer 映射信息展示

### 2.9 CMDB 资产列表页

路径建议：

- `/cmdb/assets`

主要功能：

- 统一资产列表
- 按资产类型筛选
- 按来源筛选
- 查看资产状态和标签

### 2.10 CMDB 资产详情页

路径建议：

- `/cmdb/assets/:id`

主要功能：

- 资产标准信息
- 来源信息
- 关联对象
- 更新时间

### 2.11 JumpServer 同步规则页

路径建议：

- `/jumpserver/rules`

主要功能：

- 查看规则
- 创建规则
- 编辑规则
- 启用/停用规则

### 2.12 JumpServer 同步记录页

路径建议：

- `/jumpserver/sync-logs`

主要功能：

- 查看同步历史
- 状态筛选
- 查看错误信息
- 重试失败任务

### 2.13 账单总览页

路径建议：

- `/billing/overview`

主要功能：

- 本月费用总览
- 阿里云 / AWS 占比
- 日趋势图
- Top 成本产品

### 2.14 费用分析页

路径建议：

- `/billing/analysis`

主要功能：

- 按 `env` 分析
- 按 `product` 分析
- 按账号分析
- 按资源分析
- 时间范围筛选

### 2.15 任务中心页

路径建议：

- `/tasks`

主要功能：

- 任务列表
- 任务类型筛选
- 任务状态筛选
- 查看执行日志
- 手动重试

### 2.16 审计日志页

路径建议：

- `/audit/logs`

主要功能：

- 审计日志列表
- 操作人筛选
- 模块筛选
- 时间范围筛选
- 查看详情

## 3. 页面优先级

### P0

- 登录页
- Portal 首页
- 云账号列表页
- DNS 记录列表页
- 主机列表页
- 账单总览页

### P1

- 域名列表页
- DNS 记录详情页
- 主机详情页
- JumpServer 同步记录页
- 费用分析页
- 任务中心页

### P2

- CMDB 资产列表页
- CMDB 资产详情页
- JumpServer 同步规则页
- 审计日志页

## 4. 前端公共能力

建议优先做：

- Layout 框架
- 菜单和面包屑
- 登录鉴权
- 通用表格
- 通用筛选表单
- Drawer / Modal 表单
- 图表组件
- 任务状态标记
- 空状态和异常状态页面
